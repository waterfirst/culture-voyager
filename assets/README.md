# Assets

## splash_cover.webp
매거진 커버 스플래시 이미지.
향후 GPT Image API(gpt-image-1)로 매주 새로운 커버를 생성하여 교체.

## ambient.mp3
배경 앰비언트 오디오.
Gemini Audio API로 생성하거나, 저작권 프리 앰비언트 음악 사용.
추천 소스:
- Pixabay Music (Free)
- Mixkit Ambient Sounds
- Freesound.org

## 콘텐츠별 생성 계획
- **커버 이미지**: 매주 GPT Image API로 해당 주제에 맞는 아트워크 생성
- **내레이션**: Gemini TTS로 에세이 오디오 버전 생성
- **OST/Playlist**: Spotify 임베드 또는 YouTube Music 링크
